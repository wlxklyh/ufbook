## 🎯 获取腾讯广告（GDT）ID

### 步骤 1：注册腾讯广告联盟账号

1. 访问 **腾讯广告联盟**：https://e.qq.com/ads/
2. 点击右上角「注册」按钮
3. 选择「媒体主」身份（流量主）
4. 填写资料完成注册

### 步骤 2：添加媒体/网站

1. 登录后台：https://e.qq.com/
2. 进入「流量管理」→「媒体管理」
3. 点击「添加媒体」
4. 填写您的网站信息：
   - 网站名称：UF2025 虚幻引擎嘉年华演讲总结
   - 网站地址：https://wlxkly-cloudbase-6fpwf09dd84f56e.tcloudbaseapp.com/
   - 网站分类：教育/技术
   - 提交审核

### 步骤 3：创建广告位

审核通过后（通常 1-3 个工作日）：

1. 进入「广告位管理」
2. 点击「新建广告位」
3. 创建三个广告位：

#### 广告位 1：顶部横幅
- **名称**：顶部横幅
- **类型**：横幅广告
- **尺寸**：970x90 或 728x90
- **位置**：页面顶部
- 创建后会获得一个 **广告位 ID**（如：`u1234567`）

#### 广告位 2：侧边栏
- **名称**：右侧边栏
- **类型**：矩形广告
- **尺寸**：300x250
- **位置**：页面右侧
- 创建后会获得一个 **广告位 ID**

#### 广告位 3：文章底部
- **名称**：文章底部
- **类型**：自适应
- **尺寸**：自适应
- **位置**：内容底部
- 创建后会获得一个 **广告位 ID**

### 步骤 4：获取广告代码

1. 在广告位列表中，点击「获取代码」
2. 复制广告位 ID（通常是 `u` 开头的字符串）
3. 将 ID 填入 `mkdocs.yml` 配置文件：

```yaml
tencent_ad_banner_id: "u1234567"      # 替换为您的顶部横幅 ID
tencent_ad_sidebar_id: "u2345678"    # 替换为您的侧边栏 ID
tencent_ad_article_id: "u3456789"    # 替换为您的文章底部 ID
```

## 🎨 自定义广告代码

如果您想使用其他广告平台（如 Google AdSense），可以在 `mkdocs.yml` 中配置：

```yaml
custom_ads_enabled: true
custom_ad_code: |
  <!-- Google AdSense 示例 -->
  <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
       crossorigin="anonymous"></script>
  <ins class="adsbygoogle"
       style="display:block"
       data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
       data-ad-slot="1234567890"
       data-ad-format="auto"
       data-full-width-responsive="true"></ins>
  <script>
       (adsbygoogle = window.adsbygoogle || []).push({});
  </script>
```

5. 复制广告代码

## ⚠️ 重要提醒

### 广告审核要求
- **网站必须有实质内容**：您的网站内容很丰富，应该没问题 ✅
- **网站可访问**：确保域名正常访问 ✅
- **符合广告政策**：技术教育内容通常能通过审核 ✅
- **流量要求**：有些平台需要一定的流量基础

### 收入说明
- 腾讯广告 GDT：按点击付费（CPC）或按展示付费（CPM）
- 百度联盟：类似模式
- Google AdSense：通常收益较高，但需要外网访问

### 测试广告显示
配置完成后，可以在浏览器开发者工具（F12）中检查：
1. 广告容器是否正确加载
2. 广告脚本是否执行成功
3. 是否有错误信息
