## 本地运行

### 安装 GitBook CLI

```bash
npm install -g gitbook-cli
```

### 安装依赖

```bash
gitbook install
```

### 本地预览

```bash
gitbook serve
```

然后访问 http://localhost:4000

### 构建静态网站

```bash
gitbook build
```
