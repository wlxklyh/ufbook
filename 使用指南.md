## 🔧 本地开发

### 安装GitBook CLI
```bash
npm install -g gitbook-cli
```

### 安装插件
```bash
gitbook install
```

### 本地预览
```bash
gitbook serve
```
访问：http://localhost:4000

### 构建静态网站
```bash
gitbook build
```
